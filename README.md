# Stunt

個人 IP 內容產線：把「你是誰」變成「能發的內容」。

作者：**Ash**

**線上試用 → https://hwayu000.github.io/Stunt/**

---

## 功能

**人設定位**
從自述、訪談或既有貼文，整理出核心定位、信任資產與心智模型，並訂出禁語與誠實邊界。

**人格口吻**
從你過去的貼文萃取語氣特徵——口頭禪、開場習慣、句長、語助詞比例——再用 A／B／C 三版校準到你聽得順的那一版。

**文案架構**
人設 × 口吻 × 主題，產出五段式口播逐字稿，附三秒滑掉風險檢測與一鍵矯正。

**圖卡文案**
一句主題方向，產出整套輪播圖卡的內文、版型與成品圖。19 種版型、11 種內容區塊，文字一律自動縮排進安全區，內容再多都不會溢出。背景可逐張換成自己的圖片或影片，直接匯出 PNG。

## 使用

開啟網站後以帳號進入，四個模組可以各自單獨用，也可以順著跑完一條線。

沒有綁定模型時，所有生成功能會回傳內建示範內容，並明確標示並非 AI 生成；想真的生成請到「模型設定」綁一個：

- **Google Gemini** — [aistudio.google.com/apikey](https://aistudio.google.com/apikey) 登入後按 Create API key，
  複製 `AIza` 開頭那串貼進來。Flash 系列每天有免費額度，不必綁信用卡。
- **DeepSeek** — 到 platform.deepseek.com 註冊儲值後取得 Key。中文穩定、單價低。

API Key 只存在你自己這台裝置的瀏覽器，不會上傳。
